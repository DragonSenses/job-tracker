# Instructions to run locally

1. Clone this repo (or download zip on GitHub)
2. Go to the directory the files are located

```powershell
cd /job-tracker
```

3. Install dependencies

```powershell
npm install
```

4. Run the app on localhost:3000

```powershell
npm run start
```
